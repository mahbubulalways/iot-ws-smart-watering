Smart Plant Watering System - IoT Project
Proud to present our Peripheral & Interfacing course project: Smart Garden - an IoT-based solution designed to make plant care smarter and more efficient.
Built using ESP32, WebSocket, Express.js, MongoDB, and Flutter, this system enables real-time monitoring and remote control of plant watering through a mobile application 🌍📲
✨ Key Features:
• Soil Moisture Monitoring
• Temperature Tracking
• Automatic & Manual Watering Modes
• Smart Motor Control
• Real-time Data Monitoring
• History Tracking
